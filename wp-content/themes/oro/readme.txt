== Description ==

ORO is a WordPress Theme based on Material Design Guidelines by Google. Keeping it simple, Oro is light and clean providing a great deal of versatility at the same time with custom colors, multiple layouts, featured areas, modern design and other elements to make an amazing WordPress website. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
